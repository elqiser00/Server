# نظام رفع الأفلام والمسلسلات إلى Telegram

نظام تلقائي لرفع الأفلام والمسلسلات إلى قنوات ومجموعات Telegram باستخدام GitHub Actions.

## المميزات

- رفع تلقائي للمحتوى إلى Telegram
- دعم الأفلام والمسلسلات (حتى 10 ملفات)
- تعديل أسماء الملفات تلقائيًا
- استخدام سرعة شبكة GitHub السريعة
- واجهة مستخدم عربية سهلة

## متطلبات الإعداد

1. **بوت Telegram:**
   - أنشئ بوت عبر [@BotFather](https://t.me/BotFather)
   - احصل على التوكن (Token)

2. **معرف القناة/المجموعة:**
   - للقنوات العامة: استخدم @channel_username
   - للقنوات الخاصة/المجموعات: احصل على المعرف الرقمي

## إعداد GitHub Secrets

اذهب إلى إعدادات المستودع → Secrets and variables → Actions → New repository secret

أضف الأسرار التالية:

| الاسم | الوصف |
|-------|-------|
| `TELEGRAM_BOT_TOKEN` | توكن بوت Telegram |
| `TELEGRAM_CHAT_ID` | معرف المحادثة الافتراضي |

## كيفية الاستخدام

1. افتح صفحة `index.html`
2. املأ البيانات المطلوبة:
   - معرف القناة/الجروب
   - اسم الفيلم/المسلسل
   - نوع المحتوى
   - رابط الصورة
   - رابط الفيديو (أو روابط الحلقات للمسلسل)
3. انقر على "بدء عملية الرفع"
4. تتبع العملية من تبويب Actions في المستودع
